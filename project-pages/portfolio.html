Portfolio Website

Overview

My online portfolio is built with a mobile-first strategy. I worked with media queries, a responsive grid, Sass, and semantic HTML5.

I wanted to make sure my portfolio was fully responsive and looked great no matter what device it was being viewed on. With four distinct breakpoints for screen sizes, I was able to tailor each page to display well on several devices.

Process

One of my main goals in building this site from the ground up was to focus on modular, reusable code. Using Sass was a no-brainer, as it naturally lends itself to modular thinking and repeatable styles.

An example of my focused effort on repeated styles is the individual project pages. Each page has similar styling patterns, which makes it easy for the viewer to find information and understand the page structure. It will also make it a breeze to add new project pages in the future.

Technologies

I built my portfolio with semantic HTML5, Sass, media queries, a responsive grid, and a SMACSS-inspired file structure.

Design decisions such as typography and color were made purposefully, following basic design principles.
