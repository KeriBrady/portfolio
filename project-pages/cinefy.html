Cinefy

Overview

In a four-day sprint, I worked with a team of four JavaScript full-stack developers to design and develop the Cinefy app from the ground up using AngularJS, node.js, database storage, HTML, and CSS.

Cinefy is a web application that allows movie watchers to get more out of their viewing experience by providing users with an opportunity to 'like' and later re-visit products, songs, and other fun facts from their favorite films.

Process

As the sole UI Developer, I created style tiles for branding (primarily color and typography), wireframes for site structure and layout, and led development of all HTML and CSS components in the project.

To manage tasks, my team used Trello boards. Every morning we would hold a quick stand up meeting to communicate what we had worked on, what we were going to work on that day, and if there was anything blocking our work.

Challenges

A huge challenge in designing and building Cinefy was figuring out how to showcase its intended user experience without a full native phone app. By collaborating with the JavaScript developers, I designed a demo page for the site that clearly illustrated the timed notification functionality of the app and how it would display on a mobile device as a movie is being watched.

Technologies

Cinefy's user interface was built with semantic HTML and modular CSS, organized with a SMACSS file structure.

The interactive features were built by the JavaScript developers using an AngularJS front-end, a Node.js back-end that utilizes a RESTful Express API to route user requests, and a MongoDB persistence layer.
