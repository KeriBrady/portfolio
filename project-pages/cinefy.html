Cinefy

Cinefy is a web application that allows movie watchers to get more out of their viewing experience by providing users with an opportunity to like and later re-visit products, songs, and other fun facts from their favorite films. As the sole UI Developer, I created style tiles for branding (primarily color and typography), wireframes for site structure and layout, and led development of all HTML and CSS components in the project. Cinefy was created with an AngularJS front-end, a Node.js back-end that utilizes a RESTful Express API to route user requests, and a MongoDB persistence layer.
