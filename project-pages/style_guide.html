Living Style Guide

Overview

This project required building a living style guide for an existing website, using their existing module and element styles.

The idea is based off of style guides like Primer and Mail Chimp's style guide, and would hypothetically be something for UI developers at the website's offices to use for reference when building out pages.

Process

I chose to base my style guide on Peach, a Seattle-based company that offers cost-effective food delivery service to companies in the Seattle area. I used their website for inspiration in styling, but deviated when necessary to follow HTML best practices.

For example, Peach uses all six heading tags throughout their site with multiple styles. However, best practice is to use heading tags to indicate importance of content rather than styling decisions. For this reason, I stripped the heading tags down to a simple font size range with the understanding that specific styling will be defined on a case-by-case basis.

Technologies

I built this style guide using the Node.js platform with an Express framework, created the views with Jade, styled with Sass and CSS, and used the Jeet grid system.
